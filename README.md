# India's Telecom Landscape: State-wise Penetration and Growth Analysis

**Submitted by:** Tanmay Talmale | Reg. No. 12215422 | B.Tech CSE | LPU

## Overview
Analysis of 15 years (2008–2022) of TRAI state-wise telecom subscriber data to identify growth trends, rural vs urban penetration, and top-performing markets.

## Files
- `Telecom.py` — Python analysis script
- `Table_3_1.csv` — TRAI dataset (Table 3.1)
- `Report.docx` — Full seminar report

## How to Run
```bash
pip install pandas matplotlib seaborn
python Telecom.py
```

## Tools Used
Python 3, pandas, matplotlib, seaborn

## Data Source
TRAI (Telecom Regulatory Authority of India) — Public Dataset
